# Data

This folder documents curated mutation datasets and feature matrices used in the protPheMut analysis.

The full curated dataset contains 2,114 missense mutations across six oncoproteins:

| Protein | Mutations | NDD | Cancer |
| --- | ---: | ---: | ---: |
| MEK1 | 216 | 23 | 193 |
| MEK2 | 218 | 23 | 195 |
| PI3Kalpha | 782 | 154 | 628 |
| PTEN | 674 | 220 | 454 |
| RAS | 51 | 2 | 49 |
| SHP2 | 173 | 68 | 105 |
| Total | 2,114 | 490 | 1,624 |

Curated mutation data are also available through the protPheMut web server: http://netprotlab.com/protPheMut/
